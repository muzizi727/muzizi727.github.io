MySQL基础

SQL的分类

SQL程序语言有四种类型，对数据库的基本操作都属于这四类：

  1. DDL（Data Definition Language）：数据库定义语言，用来定义数据库、表、字段
  2. DML（Data Manipulation Language）：数据操纵语言，用来修改数据库中数据
  3. DQL（Data Query Language）：数据库查询语言，用来查询数据库中的数据
  4. DCL（Data Control Language）：数据控制语言，用来控制数据库对象的权限管理、事务等
